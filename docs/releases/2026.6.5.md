# Therion Studio 2026.6.5

## Highlights

- Added Mapiah background transform support for TH2 map editing, including transform metadata parsing, editing, undo/redo, and export-oriented metadata updates.
- Improved raster background rendering so image layers stay sharp while zooming instead of being limited by the initial display resolution.
- Hardened restored Mapiah raster backgrounds so metadata-backed layers ignore stale session scale, rotation, and position values.
- Simplified the Selection inspector layout around a single Options panel.
- Improved map rendering for inline line:subtype styling and point-label -align placement.
- Fixed template for New Project from Template.
- Fixed Smart Area so confirming a candidate returns the Selection inspector to the normal selected-object state instead of leaving a stale New Area panel behind.
- Fixed the Map inspector so selecting an object in `Objects` and switching back to `Selection` shows the symbol preview.

## Known Limitations

- XTherion can open TH2 files with Mapiah rotated or scaled background metadata, but it does not display those transformed backgrounds. If such a file is edited and saved in XTherion, XTherion may rewrite many otherwise unchanged line objects, so users who care about tidy version-control diffs should prefer editing those files in Therion Studio or Mapiah.
