# Therion Studio 2026.6.6

## Highlights

- Added the first version of `.lox` 3D viewer, including a neutral loader and scene model, GPU-backed Qt Quick viewport, camera controls, layer visibility, measurement support, station hover details, and model coloring by altitude or a solid color.
- Added validator warnings with explicit `Apply Fix` cleanup for empty `line` and `area` objects inside scraps.
- Improved Raw editor completion and validation responsiveness by reusing cached Therion source snapshots and logical source projections.
- Hardened Linux, Windows, and macOS build/package workflows for the Qt Quick/QML-backed 3D viewer surface, including Linux package dependency alignment and deterministic sample-data-dependent tests.
- Fixed Map inspector numeric fields so partially typed values are not committed after a single digit while editing spin boxes.
- Fixed dense TH2 map selection so hidden line control handles do not steal clicks from visible nearby paths.
